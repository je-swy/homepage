# homepage

My first personal website. After all, it is just a résumé.

🔗 **Live:** [je-swy.github.io/homepage](https://je-swy.github.io/homepage)

---

## About

Personal résumé page built with plain HTML and CSS — no frameworks, no dependencies. Covers work experience, education, skills, and contact info.

## Tech stack

- HTML5
- CSS3 (Flexbox, CSS Variables, transitions)
- Vanilla JavaScript

## Features

- Responsive layout (mobile-friendly)
- Hover animations on contacts, avatar, and headings
- Links to LinkedIn, GitHub, and CV

## 🥚 Easter egg

There's a hidden surprise somewhere on the page. If you know the classic cheat code — you know what to do.

## Run locally

```bash
git clone https://github.com/je-swy/homepage.git
cd homepage
open index.html
```
